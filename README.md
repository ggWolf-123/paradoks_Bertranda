# The Bertrand Paradoks

### ENG

The Bertrand paradox is one of the most famous paradoxes in all of mathematics. 

This project aims to try to create a classifier of generated strings depending on the method of their draw.

To achieve this, artificial intelligence models based on a random forest (model_las_losowy.ipynb) and a neural network (model_sieć neural.ipynb) have been created.

To create the datasets, the "generowanie_danych.ipynb" file was used, which for each of the three methods randomizes 1200000 chords. 

This is a bachelor's project for bachelor's studies in artificial intelligence at the KUL (Catholic University of Lublin), for this reason the content of these files and the files themselves are in Polish.

The "Untitled-1.ipynb" file is used as a notepad.

After submitting the bachelor's degree, a pdf file containing it will also be sent here.

### PL

Paradoks Bertranda to jeden z najbardziej znanych paradoksów w całej matematyce. 

Projekt ten ma za zadanie spróbować stworzyć klasyfikator wygenerowanych cięciw zależnie od metody ich wylosowania.

Aby to osiągnąć zostały stworzone modele sztucznej inteligencji oparte na lesie losowym (model_las_losowy.ipynb) oraz sieci neuronowej (model_sieć neuronowa.ipynb).

Do stworzenia zbiorów danych został wykorzystany plik "generowanie_danych.ipynb" który dla każdej z trzech metod losuje po 1200000 cięciw.
